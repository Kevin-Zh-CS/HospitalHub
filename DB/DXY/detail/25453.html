简要：

英文名称：stool color.

别称：宝宝大便颜色、婴儿大便颜色。

就诊科室：儿科、小儿消化内科。

哪些是正常的宝宝大便颜色？

孩子的大便有各种颜色，包括不同深浅的棕色，绿色或黄色的大便。通常情况下，这些颜色都是正常的，并且这些颜色的变化并不表示孩子有任何疾病。

在某些情况下，大便颜色可以为胃肠道或肝脏问题提供重要线索。

大便可见哪些颜色？

不同深浅的棕色、绿色或黄色的大便，或食物颜色便，或血样便、白陶土样便、黑便等。

大便的正常颜色

  1. 胎便：新生儿出生后 3 天内解胎便，呈现黑绿或是深绿色，无臭味。是上皮细胞，浓缩的消化液，以及胎儿时期吞入的羊水组成的。

  2. 纯母乳喂养儿的大便：未添加辅食的母乳喂养儿粪便呈现黄色或是金黄色，为膏状或是带有少许黄色粪便颗粒，偶尔稍稀，而略带绿色，不臭，有酸味。

    人乳喂养儿大便偏酸性，可因肠道氧化性细菌的作用，可将大便中黄色的胆红素氧化为淡绿色的胆绿素，使大便偏绿。

    有时后乳摄入不足，奶中的脂肪含量少，可导致绿色泡沫便，这个跟脂肪少肠蠕动增快，乳糖未被消化就进入了肠道有关。

  3. 人工喂养儿的粪便：牛、羊乳喂养的婴儿，粪便是淡黄或是深、灰黄色 / 棕色，比较干稠，易混有白色酪蛋白凝块，混合均匀时可导致大便呈浅米色或蛋花色。

    人工喂养儿大便偏碱性，大便中胆红素会还原为无色的的粪胆原，故大便颜色较淡。

    部分婴儿喝配方奶，特别是水解配方时大便会呈绿色，可能跟肠蠕动增快（饥饿、乳糖或食物的刺激，或脂肪、蛋白质比例低时可出现）、配方中铁的影响等有关，属于生理现象。

  4. 混合喂养儿大便：混合喂养儿，大便颜色可介于纯母乳和人工喂养的颜色之间。

  5. 添加辅食后：大便多呈现褐色，深浅不一，同时受食物颜色影响：

    • 若添加蔬菜和水果，可见绿色（食物色），为少许菜泥排出；
    • 如进食西红柿、西瓜、红心火龙果，可有红色（食物色） 大便排出；
    • 如果进食蓝莓之类，还会有蓝黑色（食物色） 大便，常常被家长和医生误会为果酱样便。

  6. 如果服用铁剂，或者吞咽了血液（妈妈乳头皲裂出血），大便则有可能呈黑色。

    所以黄色、棕色、褐色或绿色都可以是正常的大便颜色，某些特殊情况还有浅米色、深绿色和黑色。如果食物 / 药物（铁）成分没有被完全消化，大便还可以有点状、条状或全部大便呈黑、白、紫、红、蓝、绿色等颜色。

异常的大便颜色

  1. 绿色便：有可能是肠道蠕动增快，比如乳糖不耐受、各种肠炎，常见如细菌 / 病毒性肠炎、过敏性肠炎等，常常伴有腹泻。

  2. 浅 / 灰白色、陶土样便：可能为肝胆有阻塞，可见于先天胆道闭锁、胆道囊肿，或后天的感染、压迫等。

  3. 黑色便：可能为胃肠道出血。

  4. 红血丝便：

    • 大便较干燥，表面红血丝：可能为便秘肛裂或是直肠息肉，血管畸形、婴幼儿痔疮比较罕见。
    • 大便稀，或伴黏液：常见于病毒、细菌感染，或食物过敏，其它炎症导致的肠病少见。

  5. 鲜血便：常见于食物过敏、病毒或细菌感染，或其它原因导致的直肠结肠炎症（假膜性结肠炎、抗生素相关性腹泻等），症状严重的需要考虑肠套叠、溶血性尿毒症综合征、下消化道出血等。

  6. 果酱样便：特别是伴有精神不好、呕吐、阵发性哭闹或腹胀硬时，要考虑有肠套叠。

  7. 粘液、脓血便：应考虑有肠道感染，常见是细菌感染，明显腹痛者考虑痢疾。

综上，如果出现这些异常的大便颜色，或伴的异常症状表现时，建议及时到医院就诊。
